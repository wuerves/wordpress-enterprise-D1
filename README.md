# wordpress-enterprise-D1
这是一款wordpress响应式企业主题，响应速度快，seo优化，这套主题设计非常美观现代清爽，响应式布局，能够在移动设备上访问。在功能方面，主题支持WooCommerce插件实现商城功能，WPML插件实现多语言功能，支持页面生产插件实现页面布局自定义。
